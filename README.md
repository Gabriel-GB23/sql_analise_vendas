# sql_analise_vendas
Este projeto contém uma série de consultas SQL desenvolvidas durante meus estudos de Banco de Dados e Análise de Dados.
O objetivo foi explorar uma base de dados de vendas utilizando SQLite, aplicando conceitos como:

JOIN entre múltiplas tabelas
Funções de agregação (COUNT, SUM)
Agrupamentos com GROUP BY
Subqueries
Filtros com WHERE
Funções de data (strftime)
Cálculo de percentuais
Análise temporal de vendas
